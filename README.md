# WOW Medical

Медичний кабінет для дитячого табору WOW CAMP.  
Швидкий пошук дитини, автоматичне заповнення даних та запис результатів огляду.

## Структура проєкту

```
├── index.html          # Головна сторінка
├── css/
│   └── style.css       # Стилі (адаптивний дизайн)
├── js/
│   ├── config.js       # Конфігурація (URL, шляхи, налаштування)
│   ├── api.js          # Інтеграція з Google Apps Script
│   └── app.js          # Основна логіка (пошук, форма, валідація)
├── data/
│   └── children.json   # Тестові дані дітей
├── assets/             # Зображення, іконки тощо
└── README.md
```

## Функціонал

- **Пошук дитини в реальному часі** — за ім'ям, прізвищем, загоном або тім-лідером
- **Автозаповнення** — після вибору дитини автоматично заповнюються: ПІБ, загін, кімната, дата народження, вік, тім-лідер, телефон батьків
- **Форма огляду** — температура, скарги, надана допомога, призначення, чекбокс «Повідомлено батьків»
- **Збереження** — виведення всіх даних у `console.log()`
- **Google Apps Script** — підготовлено функцію `saveVisit(data)` для відправки POST-запиту
- **Адаптивний дизайн** — коректне відображення на телефоні, планшеті та комп'ютері

## Як запустити

1. Відкрити `index.html` у браузері
2. Або запустити локальний сервер (рекомендовано, щоб уникнути CORS-помилок при завантаженні JSON):

```bash
npx serve WOW_Medical
# або
python3 -m http.server 8080 --directory WOW_Medical
```

## Як підключити Google Apps Script

1. Відредагувати `js/config.js` — вказати URL веб-застосунку:

```js
googleScriptURL: 'https://script.google.com/macros/s/AKfycb...your-id.../exec'
```

2. Функція `saveVisit(data)` у `js/api.js` автоматично відправлятиме POST-запит із даними форми.
